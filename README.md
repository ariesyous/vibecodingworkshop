# Vibe Coding Workshop: Defying Gravity

Welcome to the **Vibe Coding Workshop**! This repository contains all the resources you need to participate in our AI-assisted hackathon, where we use **Gemini CLI** and **Google Antigravity** to build applications at the speed of thought.

## 🚀 Workshop Overview

In this workshop, we're "defying the gravity" of traditional coding. Instead of getting bogged down in boilerplate and syntax, we leverage agentic AI to handle the heavy lifting while we focus on design, architecture, and "the vibe."

### Tools of the Trade
- **Gemini CLI**: A powerful AI agent that runs directly in your terminal.
- **Google Antigravity**: An AI-powered IDE built for agent-assisted development.
