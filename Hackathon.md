# Defying Gravity: An AI-Assisted Hackathon

Welcome to the **"Defying the Gravity of Traditional Coding"** hackathon! 

Today, we are using the **Gemini CLI** and the concept of **Antigravity** to build faster, smarter, and lighter. We'll let AI handle the "heavy lifting" (syntax, boilerplate, and structure) while you focus on the "navigation" (ideas, architecture, and fun).

All projects will utilize free, open data from the [Public APIs](https://github.com/public-apis/public-apis) repository.

Choose a track below based on your experience level and get building!

---

## Track 1: The "AI Co-Pilot" (New to Programming)

**Goal:** Practice "Prompt Engineering as Code." Use Gemini CLI to generate a complete HTML/CSS/JS or simple Python script. Your challenge is learning how to iterate and refine your instructions to the AI.

### Option 1: The "Zero-G" Fact Generator
*   **API:** [RandomDuck](https://random-d.uk/api)
*   **The Project:** A single-page web app.
*   **Antigravity Twist:** Prompt Gemini to: *"Build a web page that fetches a random duck picture and a random fun fact. Make the CSS 'space-themed' and make the images slowly float around the screen like they are in zero gravity."*

---

## Track 2: The "Systems Architect" (Experienced Programmers)

**Goal:** Build robust logic, handle data serialization, edge cases, and use Gemini CLI as a paired programming partner to solve complex problems and write complex UI faster.

### Option 1: The "Mission Briefing" Terminal
*   **API:** [Launch Library 2](https://thespacedevs.com/llapi)
*   **The Project:** A robust, visually appealing Python or Go CLI tool.
*   **Antigravity Twist:** The API returns a massive payload of data for upcoming space launches. The challenge is to parse this, handle pagination, and use Gemini CLI to help write a beautiful terminal UI (using libraries like `Rich` in Python) that displays a countdown timer, weather conditions, and mission objectives.

### Option 2: The Quantum "Bargain Hunter"
*   **API:** [CheapShark](https://www.cheapshark.com/api) (PC Game Prices)
*   **The Project:** A background script or complex aggregator.
*   **Antigravity Twist:** Build an engine that takes a local JSON file of "wishlisted" games, queries the CheapShark API, sorts the data by the best deals, and then generates a stylized HTML "Action Report." Use Gemini to rapidly write complex sorting algorithms or regex to clean up the data.

### Option 3: The Academic "Abstractor"
*   **API:** [arXiv](https://arxiv.org/help/api/user-manual) (Physics/Math papers)
*   **The Project:** An automated data pipeline.
*   **Antigravity Twist:** Query the arXiv API for recent papers tagged with "gravity" or "propulsion." Because the arXiv API can return dense XML, use Gemini to help parse the XML into a clean JSON structure, and then extract the abstracts into a local SQLite database or a Markdown summary file.

### Option 4: The Art Remix Pipeline
*   **API:** [Art Institute of Chicago](https://api.artic.edu/docs/)
*   **The Project:** A multi-step data processing tool.
*   **Antigravity Twist:** Pull metadata for historical artworks (which are in the public domain). Build a tool that strips the "heavy" historical descriptions and uses Gemini to procedurally generate a new, sci-fi/cyberpunk description for the art piece, saving the new "remixed" gallery locally.

---

## Kickoff: The Ultimate "Antigravity" Trick

To start the event, open your terminal and run:

```bash
python -c "import antigravity"
```

Have fun, and let the AI do the heavy lifting!
